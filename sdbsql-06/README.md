# Домашнее задание к занятию «Репликация и масштабирование. Часть 1»


### Задание 1

На лекции рассматривались режимы репликации master-slave, master-master, опишите их различия.

*Ответить в свободной форме.*

## Ответ 1

При репликации master-slave master является основным сервером БД, куда поступают все данные, а slave - вспомогательный сервер БД, который копирует все данные с мастера.
А при master-master репликации каждый сервер выполняет роль мастера и слейва одновременно.
